# Open3D Python Runtime

LRP Python runtime for heavy 3D processing operations.

## Setup
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Capabilities
- Blender-based format conversion (40+ formats)
- pythonOCC for STEP/IGES/BREP operations
- CadQuery for parametric CAD
- Open3D for point cloud processing

## Usage
Started automatically by LRP orchestrator via stdio bridge.
